# showup

Midnight primary, gold secondary. Refined neutral palette using `midnight-*` for the foreground scale instead of pure gray.

## Identity

- **Primary font:** Inter (fallback: Arial, system-ui)
- **Primary color:** `midnight-500`
- **Secondary color:** `gold-500`
- **Type scale:** standard (16/14/12)
- **Label weight:** 600
- **Foreground scale:** `midnight-800` (default), `midnight-300` (muted) — slightly warmer than the gray-based scale used by wireframe / klub.

## When to use

The ShowUp product surface — event / experience-management features.

## Switch on

```html
<html data-brand="showup">
```

```ts
document.documentElement.dataset.brand = 'showup'
```
